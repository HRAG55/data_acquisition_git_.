Project Overview

This project demonstrates the process of data acquisition, exploration, and version control using Git and GitHub. The dataset used is the historical Titanic dataset publicly available from Kaggle. The main objectives are:

Importing and exploring a dataset using Python and Google Colab.

Visualizing key features and deriving insights from the data.

Applying ethical and legal considerations when handling datasets.

Managing the project using Git for version control and GitHub for collaboration.
